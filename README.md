# Assistant Manil704 - PWA pour PWABuilder

Cette version est prête pour PWABuilder.

## Fichiers inclus

- `index.html`
- `manifest.json`
- `service-worker.js`
- `logo.png`
- `icons/`

## Installation sur GitHub Pages

1. Envoyez tous les fichiers à la racine du dépôt GitHub.
2. Allez dans `Settings`.
3. Allez dans `Pages`.
4. Choisissez `Deploy from a branch`.
5. Branche : `main`.
6. Dossier : `/ root`.
7. Ouvrez l'adresse GitHub Pages générée.

## Utilisation avec PWABuilder

1. Copiez l'adresse publique de votre GitHub Pages.
2. Ouvrez PWABuilder.
3. Collez l'adresse du site.
4. Lancez l'analyse.
5. Générez le package Android, Windows ou autre.

## Important

Cette version demande la clé API Mistral au démarrage.
Elle est pratique pour tester.

Pour un vrai site public, il est préférable d'utiliser un serveur sécurisé pour ne pas exposer la clé API dans le navigateur.
